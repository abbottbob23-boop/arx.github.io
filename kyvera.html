Release notes

v0.1.0 Initial release 2025

Kyvera — First build

The first version of Kyvera establishes the core platform — a self-hosted server management suite that brings together everything you need in one clean install.

  • Media server — stream your own movies and TV series in full quality across your local network
  • Network-wide ad blocking — DNS-level blocking across every device connected to your network
  • Personal cloud storage — store, access and share your files without any third party cloud service
  • Self-hosted VPN via WireGuard — secure remote access to your Kyvera server from anywhere
  • Theo AI assistant — locally hosted AI for server monitoring, diagnostics and natural language control
  • Server dashboard — real-time monitoring of CPU, RAM, GPU, storage and network from one interface
